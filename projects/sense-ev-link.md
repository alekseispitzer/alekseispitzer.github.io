# EV Link Vehicle Experience – Sense

**Role:** Senior iOS Engineer  
**Tech:** Swift, SwiftUI, TCA, Combine, async/await, SPM  
**Type:** Production feature (2025)

## Overview
Contributed to the development of the EV Link Vehicle Experience for Sense, enabling users to securely connect their electric vehicles, sync data, and prepare for upcoming managed charging capabilities.

## Key Contributions
- Built SwiftUI flows using **TCA** with modular reducers and composable navigation.
- Implemented account linking & OAuth flows for multiple EV OEMs.
- Developed reusable UI components aligned with the Sense design system.
- Improved EV polling performance and reduced data latency.
- Worked closely with Android to ensure cross-platform parity.
- Delivered features following strict release, QA, and feature-flag processes.

## Impact
Enabled a critical first step in Sense’s multi-phase EV Managed Charging roadmap and improved reliability of the EV sync experience.
