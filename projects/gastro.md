# Gastro – Restaurant Discovery App

**Role:** Solo iOS Engineer  
**Tech:** UIKit, Google Places API, Coordinator pattern, Core Location

## Overview
A personal iOS project inspired by OpenTable and Google Places, designed for discovering restaurants, filtering by cuisine, and managing favorites.

## Key Features
- Custom UI components (search bar, selected location header, filter views).
- Advanced search using **Google Places API** with location bias + routing parameters.
- Custom favorites engine with local persistence.
- Coordinator-based navigation for clean, testable routing.

## Impact
Demonstrates end-to-end product ownership, from UX to API integration to App Store-ready UI.
